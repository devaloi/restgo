# Changelog

All notable changes to restgo are documented here.

## [0.2.0] - 2026-02-20

### Added
- Request validation middleware
- OpenAPI spec (hand-written)
- GitHub Actions CI pipeline

## [0.1.0] - 2026-02-18

### Added
- User auth with JWT (no external auth libraries)
- Articles CRUD with pagination
- Middleware: logging, recovery, CORS, rate limiting
- PostgreSQL with raw `database/sql` (no ORM)
- Database migrations
- MIT License
